# MIYO — Personal Finance App

> Take control of your money. Track income, expenses, savings, 
> and subscriptions in one place.

🔗 **Live Demo:** [miyo-web.vercel.app](https://miyo-web.vercel.app)

---

## ✨ Features

- 🔐 **Authentication** — Secure register & login system
- 📊 **Dashboard** — Real-time overview of income, expenses and balance
- ➕ **Quick Entry** — Add income, expenses or savings in one click
- 📋 **Movements** — Full transaction history with filters
- 🔔 **Subscriptions** — Track recurring subscriptions with expiry alerts
- 💰 **Savings** — Individual savings goals with progress tracking
- ⚙️ **Settings** — Multi-currency support and account preferences

---

## 🛠 Tech Stack

| Layer | Technology |
|---|---|
| Frontend | Next.js 14, TypeScript, Tailwind CSS |
| Backend | NestJS, TypeScript |
| Database | PostgreSQL + Prisma ORM |
| Auth | JWT |
| Deploy | Vercel |

## 🚀 Run Locally

```bash
# Clone the repo
git clone https://github.com/j4mLoz/miyo.git

# Install dependencies
cd miyo
npm install

# Set up environment variables
cp .env.example .env

# Run development server
npm run dev
```
